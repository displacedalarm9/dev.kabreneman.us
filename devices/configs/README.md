# devices/configs/

XML configuration files for UNISYS nodes, migrated from `kabreneman.us/configs/` and `KABDMSV2/`.

All files carry a UNISYS_IMPORT_RECORD metadata header with OriginalLocation.

| File | Node ID | Role | Source |
|------|---------|------|--------|
| `config_NODE-OPS-001_Legion5Gen10.xml` | NODE-OPS-001 | Primary Workstation (Legion 5 Gen 10) | kabreneman.us |
| `config_NODE-VR-002_LegionPro5Gen10.xml` | NODE-VR-002 | Gaming/VR (Legion Pro 5 Gen 10) | kabreneman.us |
| `config_NODE-ARC-003_CustomArchival.xml` | NODE-ARC-003 | Archival Storage (Custom NAS) | kabreneman.us |

See `docs/standards/configuration-standards.md` for XML format standards.
