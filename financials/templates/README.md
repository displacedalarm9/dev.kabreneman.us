# financials/templates/

Financial planning and review templates migrated from `kabreneman.us/templates/`.

All files carry a UNISYS_IMPORT_RECORD metadata header with OriginalLocation.

## Template Categories

### Review Templates
| File | Purpose |
|------|---------|
| `daily.md` | Daily financial position tracking |
| `weekly.md` | Weekly status review |
| `monthly.md` | Monthly progress review |
| `quarterly.md` | Quarterly phase assessment |

### Report Templates
| File | Purpose |
|------|---------|
| `variance.md` | Variance analysis |
| `status.md` | System status report |
| `reports.md` | Report generation guide |
| `report/weekly.md` | Weekly status report (OPERCAP1 format) |

### Data Templates
| File | Purpose |
|------|---------|
| `accounts.md` | Account position tracking |
| `utilities.md` | Utility statement analysis |
| `obligations.md` | Obligation tracking |
| `distribution.md` | Payment allocation |
| `data/subscriptions.md` | Subscription tracking |

### Planning Templates
| File | Purpose |
|------|---------|
| `sidu.md` | Schedule for Ideal Discretionary Usage |
| `investment.md` | Investment tracking |
| `opercap1.md` | OPERCAP1 legacy report templates (archived) |
| `index.md` | Template library index |
