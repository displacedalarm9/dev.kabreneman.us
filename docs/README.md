# docs/

Canonical context documents, system guides, and instructional notes for the UNISYS system.

This directory also holds content migrated from `kabreneman.us` and `KABDMSV2`. All migrated files carry a `UNISYS_IMPORT_RECORD` metadata block.

## Subdirectories

| Subdirectory | Contents |
|---|---|
| `procedures/` | Operational procedures (changeover, emergency, matrix, schedule, audit) |
| `standards/` | Coding, documentation, configuration, and file organization standards |
| `archive/` | Historical and archived documents |

## Key Documents

| File | Description |
|------|-------------|
| `automation-guide.md` | Work documentation automation guide |
| `filename.md` | File naming conventions (WORKCAP) |
| `folder-audit.md` | Folder structure analysis |
| `history.md` | System terminology and naming history |
| `report-standards.md` | Report generation standards |
| `file-management.md` | File management guidelines (from KABDMSV2) |
| `docsys-lifecycle-stages.md` | DOCSYS artifact lifecycle stages |
