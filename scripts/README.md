# scripts/

Automation scripts, asset identity generators, and acquisition workflows for the UNISYS system.

## Files

| File | Description | Source |
|------|-------------|--------|
| `fix_filenames.py` | Renames WORKCAP review files to standard naming convention | kabreneman.us |
| `workcap_analyzer.py` | Analyzes WORKCAP markdown files for structure, metrics, and cross-references | kabreneman.us |

All scripts carry a `UNISYS_IMPORT_RECORD` metadata header with `OriginalLocation`.
