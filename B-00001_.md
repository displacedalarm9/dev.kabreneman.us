# B-00001_ — Outstanding Document Index Generation Brief

## Purpose

This brief instructs GitHub Copilot to generate the "Outstanding Document Index" — a registry of all reserved but uninstantiated identity-bearing documents across all DOCSYS classes (X, G, M, E, R, S, CFG, B, L, P).

## Scope

This brief applies to a single generation task. The output is a standalone Markdown registry file. The brief does not govern the documents listed within the index — only the act of generating the index itself.

## Instructions

- Produce a single Markdown file titled "Outstanding Document Index."
- Include sections for each class: X, G, M, E, R, S, CFG, B, L, P.
- Under each section, list all reserved IDs that do not yet have an instantiated Page.
- Do NOT generate content for the documents themselves; only list the IDs and their intended titles.
- Use bullet lists, not tables.
- Maintain strict DOCSYS identity formatting: CLASS-NNNNN_.
- Do not assign TSNs.
- Do not create PMFs.
- Do not generate headers or metadata blocks within the index.
- The output file is a registry only.

## Output

File: `L-00001_.md`
Location: Repository root
